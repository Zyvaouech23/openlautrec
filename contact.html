Questions fréquentes

Comment installer OpenLautrec sur mon ordinateur ?
Téléchargez le fichier d'installation depuis la page de téléchargement, décompressez l'archive et exécutez OpenLautrec-Setup.exe. Suivez les instructions à l'écran. Un guide d'installation détaillé est disponible dans la documentation.
OpenLautrec est-il compatible avec Windows 11 ?
Oui, OpenLautrec est entièrement compatible avec Windows 10 et Windows 11. Il fonctionne également sur macOS 10.14+ et les distributions Linux principales (Ubuntu, Debian).
La reconnaissance vocale fonctionne-t-elle sans connexion Internet ?
Certaines fonctionnalités de reconnaissance vocale nécessitent une connexion Internet pour fonctionner de manière optimale. Cependant, un mode hors ligne basique est disponible avec des capacités limitées.
Puis-je personnaliser les polices et les couleurs dans le mode dyslexie ?
Oui ! OpenLautrec offre de nombreuses options de personnalisation pour le mode dyslexie, incluant le choix des polices, la coloration des syllabes, l'espacement et les contrastes. Accédez aux paramètres d'accessibilité depuis le menu principal.
Comment puis-je signaler un bug ou suggérer une amélioration ?
Utilisez le formulaire de contact ci-dessus en sélectionnant "Signaler un bug" ou "Suggestion de fonctionnalité" comme sujet. Vous pouvez également ouvrir une issue directement sur notre page GitHub pour les utilisateurs plus techniques.
OpenLautrec est-il vraiment gratuit ?
Oui, OpenLautrec est entièrement gratuit et open-source sous licence MIT. Il n'y a aucun coût caché, aucun abonnement et aucune publicité. Le projet est développé pour servir la communauté éducative.
